# E-Commerce Customer Churn Analysis (End-to-End Power BI Project)

## 📌 Project Overview
This project focuses on analyzing customer churn for an e-commerce platform to identify the key factors driving customer attrition. By analyzing customer demographics, behavior, and transactional data, this dashboard provides actionable insights to help the business improve customer retention.

## 📊 Key Insights & Metrics
- **Total Customers:** 5.63K | **Churned Customers:** 948 | **Overall Churn Rate:** 16.84%
- **New Customer Risk:** Churn rate is exceptionally high (**53.54%**) within the first month of tenure.
- **Customer Complaints:** Customers who raised complaints have a massive churn rate of **31.67%**, compared to only 10.93% for those who didn't.
- **Preferred Categories:** Mobile Phone users show the highest tendency to churn (**27.40%**).
- **Logistics Impact:** A significant spike in churn (**up to 33.66%**) is observed when the warehouse-to-home distance is between 5 to 30 units.

## 🛠️ Tech Stack & Tools Used
- **SQL:** Data cleaning, handling missing values, and data transformation.
- **Power BI:** Data modeling, DAX measures, and interactive dashboard design.
